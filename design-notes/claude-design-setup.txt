HOW TO ADD THIS DESIGN SYSTEM TO CLAUDE DESIGN
================================================

You are on the "Set up your design system" screen. Here is exactly what to do.

------------------------------------------------

Copy and paste this in:

Either one works. The computer option needs no login.

------------------------------------------------
3) FONTS / LOGOS / ASSETS  (optional)
------------------------------------------------
No font files are needed — Inter + IBM Plex Mono load from Google Fonts.

These are inside the code folder, so the "Link code from your computer" /
GitHub option already picks them up. If you ALSO want to drag them into the
"Add fonts, logos and assets" box, select the files inside assets/brand and

------------------------------------------------
THEN

Continue / submit. Claude will read the code and build your design system.
